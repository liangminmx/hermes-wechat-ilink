# Debian/Ubuntu 23.10+ 依赖安装指南
# 由于系统使用 "externally-managed-environment" 保护，
# 不能直接用 pip install 安装系统级包

## 推荐方法 (选择一种):

### 方法1: 使用系统包管理器（最推荐）
```bash
sudo apt update
sudo apt install python3-qrcode python3-aiohttp
```

### 方法2: 创建虚拟环境（安全隔离）
```bash
# 创建虚拟环境
python3 -m venv /opt/hermes-wechat-venv

# 激活环境并安装
source /opt/hermes-wechat-venv/bin/activate
pip install qrcode[pil] aiohttp

# 使用方式 (在插件脚本中激活)
cd /opt/hermes-agent/hermes-agent-main/plugins/memory/hermes_wechat_ilink
source /opt/hermes-wechat-venv/bin/activate
python3 __main__.py --auth
```

### 方法3: 使用 pipx（独立应用）
```bash
# 先安装pipx
sudo apt install pipx
pipx ensurepath

# 安装依赖（分别安装）
pipx install qrcode[pil]
pipx inject qrcode[pil] Pillow
pipx install aiohttp
```

### 方法4: 强制安装（不推荐）
```bash
# 使用 --break-system-packages 参数
pip install qrcode[pil] aiohttp --break-system-packages
```

## 依赖详情:
- qrcode[pil] >= 7.4.2   # 用于生成二维码
- aiohttp >= 3.9.0       # 异步HTTP客户端
- Pillow >= 10.0.0       # 图片处理（已包含在qrcode[pil]中）
