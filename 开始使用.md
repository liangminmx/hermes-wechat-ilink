# 🚀 微信插件 - 超简单开始指南

## ⏱️ 1分钟速通

```bash
# 📍 第一步：进插件目录
cd /opt/hermes-agent/hermes-agent-main/plugins/memory/hermes_wechat_ilink

# 📍 第二步：扫码认证（核心！）
python3 -m hermes_wechat_ilink --auth

# 📍 第三步：在飞书使用
/wechat_send_message --to_user_id "wxid_test" --message "你好微信"
```

---

## 📋 详细步骤

### **1. 准备环境**
```bash
# 检查依赖
cd /opt/hermes-agent/
pip install qrcode[pil] aiohttp 2>/dev/null || echo "安装中..."
```

### **2. 扫码认证**
```bash
# 进入插件目录
cd hermes-agent-main/plugins/memory/hermes_wechat_ilink

# 运行认证
python3 -m hermes_wechat_ilink --auth
```
**屏幕上会显示二维码** → 🖱️ **用手机微信扫它** → ✅ **认证成功！**

### **3. 检查状态**
```bash
python3 -m hermes_wechat_ilink --status
# ✅ 应该显示"微信已认证: true"
```

### **4. 发送微信消息**
**在飞书中输入：**
```
/wechat_send_message --to_user_id "wxid_xxx" --message "Hello from Hermes"
```

### **5. 接收微信消息**
```
/wechat_get_messages
```

---

## 🔑 核心命令表

| 做什么 | 终端命令 | 飞书命令 |
|-------|----------|----------|
| **首次认证** | `python3 -m hermes_wechat_ilink --auth` | - |
| **查看状态** | `python3 -m hermes_wechat_ilink --status` | `/wechat_status` |
| **发送消息** | `python3 -m hermes_wechat_ilink --send --to "wxid" --message "内容"` | `/wechat_send_message --to_user_id "wxid" --message "内容"` |
| **接收消息** | `python3 -m hermes_wechat_ilink --receive --timeout 10` | `/wechat_get_messages` |
| **退出登录** | `python3 -m hermes_wechat_ilink --logout` | - |

## ❓ 遇到问题？

```bash

python3 -m hermes_wechat_ilink --logout
python3 -m hermes_wechat_ilink --auth
```

### **问题2：看不到二维码？**
```bash
# 安装二维码库
```

```bash

# 重启Hermes（认证后需要重载插件）
# ...重启你的Hermes Agent
```

## 🎯 核心概念

1. **一次扫码，永久使用** - 凭证保存到 `~/.hermes/wechat_credentials.json`
2. **终端认证，飞书使用** - 认证必须在终端，使用可以在飞书
3. **完全模仿wx-robot-ilink** - 体验和Node.js版一模一样
